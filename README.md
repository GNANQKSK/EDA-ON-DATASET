# 📊 Customer Purchase Analysis (Python Project)

## 📌 Overview

This project performs **exploratory data analysis (EDA)** on a customer purchase dataset to uncover patterns, trends, and insights related to buying behavior.

Using Python libraries like **Pandas** and **Matplotlib**, the project analyzes how different factors such as gender, age, city category, and occupation influence purchase amounts.

---

## 🚀 Features

The project includes the following key functionalities:

* ✅ Data loading and preprocessing
* ✅ Dataset exploration (shape, columns, missing values, statistics)
* ✅ Total and average sales calculation
* ✅ Purchase distribution visualization
* ✅ Sales analysis based on:

  * Gender
  * Age group
  * City category
  * Marital status
  * Occupation (Top 10)
  * Product category
  * Stay in current city
* ✅ Pie chart for sales contribution by gender
* ✅ Top 10 highest purchases identification
* ✅ Correlation analysis with purchase amount
